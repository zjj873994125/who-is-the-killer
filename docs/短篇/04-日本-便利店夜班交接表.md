# 便利店夜班交接表

## 故事正文

风见町车站旁的便利店每天凌晨四点交接。夜班店员要在平板上填写库存、损耗和异常情况，再由早班确认。出事前一周，交接表上多了一个名字：桥本真由。

店长说桥本真由三年前就离职了。她离职后的第二天，骑车回家时被货车撞死。可系统显示，她最近每天凌晨三点五十五分都会完成一次夜班交接，备注也很普通：饭团补货完成，热柜清洁完成，关东煮汤底更换。

新来的夜班店员宫泽不信鬼。他说有人盗用旧账号，于是凌晨守着平板。三点五十五分，平板自己亮了，屏幕上出现一行字：桥本真由请求交接。宫泽按下拒绝，便利店的自动门突然打开，门铃响了一声，外面却没人。

监控里，货架之间出现短暂阴影。不是人形，更像雨衣下摆。宫泽听见冷藏柜旁有人翻塑料袋，随后平板自动弹出交接清单。清单比平时多一项：请确认遗失物。

遗失物柜里有一枚旧工牌，照片是桥本真由。工牌背面贴着一张小票，时间是她死亡当天凌晨四点零二分，商品是一份明太子饭团和一瓶热咖啡。奇怪的是，事故报告写她三点五十八分就已经离店。

调查员查看三年前监控备份。当天凌晨，桥本真由确实下班，走到门口又折返回来。她像听见谁叫她，回到冷藏柜前，从货架后面拿出一个黑色塑料袋。监控在她打开袋子时突然中断。恢复后，她骑车离开，五分钟后出车祸。

店长承认，当年店里发生过盗摄事件。有顾客投诉试衣间被偷拍，虽然便利店没有试衣间，但员工休息室里确实发现了针孔摄像头。店长怕影响加盟资格，把摄像头和存储卡装进黑袋，打算天亮后丢掉。桥本真由发现后准备报警。

那晚有人拦住她。货车司机供述，他撞上桥本前，看见她从便利店方向冲出来，后面有一个穿店员制服的人追她。可警方没有找到追逐者，店长说那时店里只有桥本一人。

最近的异常交接表不是单纯闹鬼。有人重新启用了桥本的账号，把旧库存系统连到店内设备，试图引导警方找到三年前的黑袋。黑袋最终在冷藏柜底部夹层被发现，里面有摄像头、存储卡和一枚带血的备用钥匙。

存储卡里拍到店长和区域督导争吵。督导要求压下盗摄事件，店长把黑袋塞给桥本，让她“帮忙处理”。桥本拒绝，转身离店。视频最后，督导追了出去。

督导被带走那天，便利店停业半日。晚上重启系统，交接表上桥本真由的名字消失了。

但早班店员说，凌晨四点开门时，热柜里多了一份明太子饭团和一瓶热咖啡。小票放在旁边，价格为零。

备注栏写着：本日交接完成。

## 游戏化要点

- **核心怪谈**：死去三年的店员每天凌晨完成夜班交接。
- **真相方向**：便利店系统异常引出三年前被压下的盗摄和追逐事故。
- **关键场景**：便利店前台、冷藏柜、遗失物柜、员工休息室、交接系统。
- **关键线索**：桥本账号、三点五十五分交接、小票时间差、黑色塑料袋、针孔摄像头、带血备用钥匙。
- **检索关键词**：桥本真由、交接表、遗失物、04:02、黑袋、明太子饭团。
- **复盘问题**：为什么死人账号还能交接？桥本死亡当天为什么离店后又返回？
