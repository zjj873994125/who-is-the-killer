# 不存在的换乘站

## 故事正文

城市轨道公司每年都会处理几起寻人协查，大多和醉酒、离家出走、手机没电有关。直到白石隼人的案子送来，交通卡后台第一次出现了一个不属于任何线路的站名。

白石隼人，三十二岁，系统工程师，独居。他最后一次被人看见，是周五晚上十点四十四分，在新都线的月见町站进站。监控里他穿灰色风衣，背黑色电脑包，低头看手机，和所有赶末班车的人没有区别。

异常发生在十点五十九分。白石的交通卡记录显示，他在“白线站”完成换乘，随后刷入北环线。但轨道公司的线路图里没有白线站，调度系统也没有这座站。更怪的是，北环线当天因为轨道检修，十点半以后已经停运。

警方最初认为是后台数据错乱。技术员导出原始记录后发现，“白线站”的站点编号不是乱码，而是一个废弃编号。十七年前，地铁规划里确实有一座临时施工站，编号 S-044，名字没有最终确定，后来因为地层塌陷取消。施工队撤场时，有三名工人被困在尚未封闭的换乘通道里，官方记录称他们在七小时后获救。

白石的手机在失踪第二天短暂上线，定位仍在月见町站附近。妻子早逝后，他只有一个姐姐。姐姐交出他的电脑，里面有一份未发出的投诉邮件，标题是“月见町站站台广播错误”。邮件里写，他最近连续三晚听见末班车后还有一班列车进站，广播声很轻：“请前往白线站换乘。”

调查员调取月见町站全部监控。白石上车后，车厢摄像头一直正常，直到列车经过月见町到朝雾台之间的隧道。画面出现了十四秒雪花。恢复后，白石的座位空了，电脑包还在架子上。

列车长说，那段隧道没有停车，也没有异常制动。但运行日志显示，列车在隧道中段多消耗了四十二秒。四十二秒不足以完成停车开门，却足够一个人听见广播、站起身、走到门边。

第三天凌晨，轨道公司接到一通内部维修电话。对方自称在白线站站务室，要求补送一箱荧光灯管。录音里有低沉的风声，还有老式广播的电流声。接线员问站点位置，对方停顿很久，说：“你们不是每天都有人过来换乘吗？”

警方带着工程图进入月见町站下方的废弃检修道。通道入口被水泥封死，但水泥墙上有新鲜划痕，像有人从里面用钥匙刮过。墙体后方探测到空腔，钻孔探头伸进去时，镜头拍到一块褪色站牌，白底黑字，只剩两个字：白线。

在废弃站台尽头，他们找到白石的手机。手机电量为零，屏幕裂开，备忘录里多了一行字：不要上第四节车厢，不要听末班车后的广播。

更深处是一条没有竣工的换乘通道，地面残留脚印。脚印不止白石一个人的，有旧式工靴，也有近几年流行的运动鞋。通道墙上贴着十七年前的施工告示，告示旁边用指甲刻着很多名字。技术员用侧光拍照，发现名字后面都跟着日期，最早的是十七年前，最新的是白石失踪那晚。

那三名被困工人的档案被重新翻出。官方记录说他们获救，但医院没有收治记录，家属赔偿协议也被列为保密。老档案员终于承认，当年塌方后只找到三盏安全帽灯。为了赶工程验收，失踪被改成获救，施工站被从规划里删除。

白石并不是第一个听见广播的人。他只是第一个留下足够多记录的人。

案子最后以“地下设施事故导致失踪”结案。月见町站封闭了两天，废弃通道被灌浆填平。轨道公司更新了全部广播系统，删除了编号 S-044。

一个月后，白石姐姐收到一封没有发件人的邮件。附件是一张站台照片，照片里白石站在白色站牌下，身后排着一长队乘客。所有人都看向镜头，像在等待检票。站牌上的字很清楚：白线站。

邮件正文只有一句话：末班车后，请勿换乘。

## 游戏化要点

- **核心怪谈**：系统记录里存在一座现实中被删除的换乘站。
- **真相方向**：被掩盖的施工事故、废弃站台、自动广播系统和失踪乘客记录互相咬合。
- **关键场景**：月见町站、车厢监控、轨道公司后台、废弃检修道、白线站站台。
- **关键线索**：交通卡记录、站点编号 S-044、列车多出的四十二秒、未发投诉邮件、废弃施工告示、墙上姓名日期。
- **检索关键词**：白线站、S-044、末班车、四十二秒、第四节车厢、施工站。
- **复盘问题**：白石为什么会在没有停站记录的隧道中消失？白线站为什么会出现在交通卡后台？
