# 祠堂第十三块牌位

## 故事正文

槐安村李氏祠堂只有十二块祖牌，这是村里孩子都背得出的事。可清明前三天，守祠人开门上香，供桌上多出第十三块牌位。

牌位没有名字，只有一行小字：李氏未归人。香炉里的香灰断成三截，像被人用手指按过。守祠人吓得去叫族长，族长却说不许声张，只让人把牌位收起来。

第二天，族谱也变了。最新修订的电子族谱里多出一页，页码在十二房和十三房之间，姓名栏空白，生卒年空白，关系写着：长房嫡出。管理员说自己没有改过，后台日志显示修改账号是上一任族谱先生，而那位先生已经去世七年。

调查员来到祠堂时，村里正在准备祭祖。族长坚持说这是恶作剧。但祠堂梁上挂着的老照片里，调查员发现一个奇怪位置：所有人站成两排，中间被撕掉一块，像有个人被剪走。

村里老人提到一个名字：李照秋。她是长房女儿，四十年前考上师范，后来“跟戏班跑了”。族谱里没有她，村志里也没有她。可第十三块牌位背面，用针刻着“照秋”两个字。

线索从一口封井开始。祠堂后院有口井，被青石板压住。村里说井水犯冲，不能开。调查员发现青石板下压着一枚银发夹，发夹样式和老照片中被撕掉的人影头饰一致。

当年李照秋并不是跟戏班跑了。她发现族里挪用公粮，准备向乡里举报。族长和几名房头为了保住宗族脸面，把她关进祠堂，逼她承认与戏班私奔。她不从，混乱中跌入后院井中。事情被长辈压下，族谱删名，老照片剪影。

第十三块牌位的出现，是因为新族谱数字化。年轻管理员扫描旧族谱时，发现夹页里有一张被撕掉的女丁登记。他没有声张，私下查访老人，却在一周后溺死在村口水塘。村里说他夜钓落水，但他手机里保存着第十三块牌位照片。

牌位是谁放的？调查员起初怀疑管理员生前所为。可牌位木料是四十年前祠堂修缮剩下的阴沉木，只有守祠人知道存放位置。守祠人最终承认，是他父亲临终前交代，要在今年清明前把牌位放回去。

守祠人的父亲当年参与封井。他一辈子不敢说，只在死前留下钥匙和一封信。信中写，李照秋落井后曾敲了很久井壁，祠堂里所有人都听见了，但没人敢开井。

清明当天，村民在调查员监督下打开封井。井底淤泥中找到女性遗骨，旁边有半块木牌，上面刻着未完成的名字。牌位原来不是后来做的，而是李照秋生前给自己刻的。她知道族里要删掉她，想给自己留一个位置。

案件公开后，李氏宗族重新修谱，把李照秋写回长房。第十三块牌位正式摆上供桌，名字终于补全。

祭祖结束，香炉里的香烧得很直。只有第十三块牌位前的香灰没有倒，落成一个小小的“归”字。

## 游戏化要点

- **核心怪谈**：祠堂多出无名第十三块牌位，电子族谱也随之改变。
- **真相方向**：族谱删名、宗族遮丑、封井旧案和数字化触发重查。
- **关键场景**：李氏祠堂、族谱室、后院封井、老照片墙、祭祖现场。
- **关键线索**：无名牌位、电子族谱修改日志、被撕老照片、银发夹、临终信、封井遗骨。
- **检索关键词**：第十三块牌位、李照秋、未归人、封井、族谱先生、银发夹。
- **复盘问题**：为什么族谱会自动多出一页？第十三块牌位为什么没有名字？
