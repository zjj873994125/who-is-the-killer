# 自动售货机第七排

## 故事正文

东町公寓门口有一台蓝色自动售货机，靠近垃圾房，白天卖咖啡和矿泉水，夜里灯管发冷。住户都知道它有六排按钮，可事故发生后，监控里多出了一排。

第一名死者是公寓管理员松井。他凌晨一点巡楼，路过售货机时停下，投币，按了最下方一个按钮。监控画面很糊，只能看见出货口掉下一罐黑色饮料。松井低头看了一眼，整个人僵住。十分钟后，他从六楼天台坠下。

警方在他口袋里找到那罐饮料。罐身没有品牌，只有白色贴纸，上面打印着一行字：松井昭，死亡时间 01:18。法医确认死亡时间正好是一点十八分。

第二个拿到饮料的人是便利店夜班店员。她没有死，但精神崩溃。她说自己只是想买热咖啡，售货机最下面突然亮起第七排按钮，每个按钮下面都没有商品图，只有一个很小的数字。她按了 04，出来的罐子写着她母亲的名字。她赶回家时，母亲已经因煤气中毒昏迷。

东町公寓开始流传说法：第七排卖的不是饮料，是死亡通知。住户避开那台机器，但每天凌晨，第七排仍会在监控里亮起。机器维修员检查后台，发现销售记录每天都会多一条，商品代码为 D-7，价格为零元。

调查员发现第七排并非凭空出现。售货机内部确实有一块被拆掉的扩展面板，原本用于测试新产品。面板线路被人重新接上，控制板里写入了一个定时程序。恐怖的是，程序没有联网，却能打印出当天即将死亡的人名。

线索来自松井的巡楼日志。死亡前一周，他连续记录“售货机后方有敲击声”。垃圾房墙后曾经是公寓旧管理室，十年前因火灾封闭。火灾中死了一名女高中生，她被怀疑偷东西，管理员和几名住户把她锁在管理室等警察来。那晚售货机线路短路起火，火势烧穿墙面，女孩没能出来。

当年的管理员就是松井。他在事故报告中写，女孩自行反锁房门。但旧钥匙柜照片显示，管理室备用钥匙当天被取走，签名人是松井。

第三晚，调查员守在售货机前。凌晨一点，机器灯光闪烁，第七排亮起。按钮下面不是数字，而是调查员自己的姓。出货口吐出一罐温热的乌龙茶。贴纸上写着一个陌生名字：远藤美咲。

远藤美咲正是十年前火灾中的女孩。

罐底还有一行小字：请替我送到六楼。

调查员没有按怪谈逻辑行事，而是拆开售货机外壳。控制板背面贴着一张旧学生证，学生证已经熏黑，只剩照片轮廓。机器后方墙面有一处新补的水泥，撬开后露出旧管理室的门。门后很窄，地上有烧焦的书包、钥匙和一个手写本。

手写本记录了十年前的真相。远藤美咲不是小偷，她发现管理员用售货机隐藏住户信件和现金。松井怕事情败露，把她关进旧管理室。火灾不是意外，是售货机线路被他临时改接后短路。

所谓死亡通知并不是灵异预言。后来的两名受害者，都和当年事件有关。有人在背后操控售货机，把真相包装成怪谈，逼相关人自首或崩溃。程序打印的人名来自一份十年前的住户名单和近期健康记录。

可案子到这里并没有完全解释。操控者是谁，警方没有查到。售货机控制板里最后一次运行记录发生在旧管理室门被打开之后，时间为凌晨一点十八分。那时机器已经断电。

记录内容只有一行：D-7 商品售罄。

一个月后，公寓换了新售货机。新机器只有六排按钮。住户说，凌晨经过时，偶尔还能听见出货口里有易拉罐滚动的声音。

## 游戏化要点

- **核心怪谈**：自动售货机第七排会吐出写有死亡时间的饮料。
- **真相方向**：人为复仇和旧火灾案交织，结尾保留无法解释的断电运行记录。
- **关键场景**：公寓门口、垃圾房、旧管理室、售货机内部、六楼天台。
- **关键线索**：D-7 商品代码、死亡贴纸、巡楼日志、旧钥匙柜照片、熏黑学生证、断电运行记录。
- **检索关键词**：第七排、D-7、远藤美咲、旧管理室、01:18、售罄。
- **复盘问题**：第七排如何制造死亡预告？谁和十年前火灾有关？
