# 姜面不落地

## 故事正文

百溪镇每年冬祭都要祭祀姜神。镇上的规矩很硬：祭祀面具戴上前要请香，摘下后要挂回梁上，不能落地。落地，就要换脸。

今年冬祭结束后，戏台后台死了一个人。死者是民俗馆馆长何敬山，脸上扣着一张黑色祭祀面具。面具边缘压进皮肤，像戴了很久。可祭祀名单里没有人用这张面具。

更怪的是，后台地面没有脚印。雪天泥水很重，所有参祭人进出都留下痕迹，只有何敬山倒下的位置干干净净。面具旁边有一截红绳，红绳末端沾着香灰。

镇民说，这是姜神换脸。百溪镇供的不是外来的神名，而是本镇传下来的姜神。老人说姜神原名姜浩，生前是最早一代掌面人，死在一场冬祭之后。后来镇上不敢直呼其名，只在祭文里写作“姜神”。

何敬山去年主持修民俗馆，把一批旧祭祀面具从祠堂搬出来展览，坏了规矩。镇上把祭祀姜神时戴的面具统称为姜面，最忌讳的，就是那张黑色的黑姜面。

调查员不信神。黑姜面很重，内侧有一枚生锈铁钉，正好刺入死者颈侧动脉。凶器就是面具。问题是，谁能在满是参祭人的后台，把面具扣到馆长脸上，还不留下脚印？

线索来自一段游客视频。视频里，冬祭最后一场“驱疫”时，有个戴白姜面的参祭人走错位，从梁下经过。镜头晃动一瞬，梁上挂着的黑姜面不见了。

民俗馆登记册显示，黑姜面旧名“无口将军”，四十年前曾在一次冬祭后失踪。那年镇上发生过火灾，烧死一名祭祀掌面人。何敬山后来从外地收回这张面具，作为镇馆展品。

死者办公室里有一份未公开论文，内容是百溪姜神祭传承造假。何敬山发现，现任掌祭家族并非正统。当年真正的掌面人被排挤，火灾也不是意外。黑姜面失踪，是因为有人把证据藏在面具夹层里。

调查员拆开黑姜面，发现内层有一张发黑的布条，上面写着三个名字和一笔祭田归属。布条开头还写着一句旧话：姜浩掌面，祭田归公。四十年前，掌祭家族为了争夺祭田和祭祀名额，纵火烧死原掌面人，并把面具带走。何敬山准备公开论文，触动了他们。

凶手不是参祭人，而是负责升降布景的老舞台工。他是现任掌祭人的叔叔。冬祭前，他把黑姜面挂在梁上，用红绳连接机关。何敬山按惯例到后台检查香案时，红绳拉动，面具从梁上落下，铁钉刺入颈侧。面具“落地”的禁忌，成了杀人机关。

干净地面也有解释。何敬山倒下的位置原本铺着一块旧毡布，吸走了泥水和血迹。凶手在混乱中卷走毡布，伪造成无脚印现场。

案子破了，镇里却不愿公开。因为一旦承认姜神祭传承造假，整个民俗馆、旅游项目和家族荣誉都会崩塌。调查员坚持把证据移交，冬祭被暂停。

黑姜面作为证物封存前，工作人员发现它内侧多了一道裂纹。裂纹像嘴，向下弯着。原本“无口”的面具，像是终于开了口。

封存柜半夜报警。值班员赶到时，柜门没开，面具也在。只是面具不再面朝柜门，而是转向墙角。墙角地上，有一小撮香灰，排成两个字：还面。

## 游戏化要点

- **核心怪谈**：姜面不能落地，落地就要换脸。
- **真相方向**：祭祀禁忌被设计成杀人机关，背后是姜神祭传承造假和旧火灾案。
- **关键场景**：冬祭戏台、后台梁架、民俗馆、死者办公室、证物封存柜。
- **关键线索**：黑姜面、红绳、铁钉、游客视频、未公开论文、面具夹层布条、旧毡布、姜浩旧名。
- **检索关键词**：姜面、黑姜面、姜浩、无口将军、红绳、祭田、传承造假、旧毡布。
- **复盘问题**：无脚印现场如何形成？禁忌如何被凶手改造成机关？
