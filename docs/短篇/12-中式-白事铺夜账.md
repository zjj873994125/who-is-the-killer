# 白事铺夜账

## 故事正文

永安街有家老白事铺，门脸很窄，纸人扎得像真人。老板娘每天早上第一件事，是翻夜账。所谓夜账，是凌晨以后自动写进账本的订单。

起初她以为是儿子搞错。账本上多出一笔：寿衣一套，男款，灰蓝色，收货人赵德贵，送货时间明日午时。第二天中午，赵德贵在家中突发心梗。家属来买寿衣，指名要灰蓝色。

第二笔夜账是纸钱三刀，收货人林桂香。第三天，林桂香在菜市场摔倒去世。第四笔夜账订了一对纸童，收货人是一个还活着的孩子。

白事铺老板娘报案。她说账本晚上锁在柜里，钥匙只有自己有。监控拍到凌晨两点，柜门自己开了一条缝，一支毛笔从笔架上滚下，账本翻开。画面到这里出现雪花，恢复后，账本上已经写好新订单。

调查员检查账本。字迹不是老板娘的，也不是打印。墨迹里混着香灰。每笔订单都提前一天出现，内容准确到尺码、忌色和下葬时辰。

永安街老人说，白事铺前身是义庄。几十年前，附近拆迁，很多无主尸骨暂存在这里。后来义庄改成铺子，地下却还有一间封死的停灵室。

夜账里的收货人并非随机。他们都住在旧拆迁片区，或者家里曾参与过义庄改造。赵德贵当年是施工队长，林桂香是街道会计，订纸童的孩子，是当年开发商的孙子。

调查员在白事铺后墙找到一道暗门。门后楼梯通向地下停灵室。停灵室里没有尸体，只有一排旧木牌，木牌上没有姓名，只有编号。最里面的墙上贴着一张拆迁补偿表，表格显示，当年有十七具无主尸骨被登记为“已迁葬”。

可迁葬记录是假的。为了赶工期，施工队把无主尸骨装进编织袋，夜里运到河边掩埋。白事铺老板娘的父亲曾是义庄看守，他发现后想举报，却在一个雨夜失踪。外界说他卷款跑了。

夜账不是鬼账，而是老板娘父亲留下的旧账。调查员在停灵室砖缝里找到一本发霉账簿，记录了十七具无主尸骨的衣物特征和暂存位置。有人近期发现这本账簿，开始按当年参与者名单下单，制造死亡预告。

嫌疑指向老板娘的儿子。他承认伪造了部分夜账，为的是逼母亲公开外公失踪真相。但他只写了前三笔。后面的纸童订单不是他写的，字迹也不同。那笔订单上的孩子并没有死亡，却在当天午时发高烧，说梦见一间地下屋子，屋里有人问他：“你家欠的纸钱什么时候还？”

警方根据旧账簿找到了河边掩埋点，挖出多具遗骨，也找到老板娘父亲的遗物。他不是卷款逃走，而是被施工队灭口，和无主尸骨埋在一起。

旧案公开后，白事铺关门七日做法事。老板娘把十七块无名木牌重新写上“无名归位”，烧了纸钱和寿衣。

第七天夜里，账本又翻开了。监控没有雪花，画面很清楚。毛笔自己立起，慢慢写下一笔订单。

货品：白纸灯一盏。

收货人：看守人。

送货时间：归家路上。

第二天清晨，老板娘在铺门口发现一盏没有点燃的白纸灯。灯下压着父亲当年的钥匙，锈已经掉尽，像刚被人握过。

## 游戏化要点

- **核心怪谈**：白事铺账本会提前写出即将死亡之人的丧葬订单。
- **真相方向**：伪造夜账、义庄旧案、无主尸骨和看守人失踪构成主线。
- **关键场景**：白事铺、夜账柜、地下停灵室、旧拆迁片区、河边掩埋点。
- **关键线索**：夜账、香灰墨迹、拆迁补偿表、无名木牌、旧账簿、纸童订单、看守人钥匙。
- **检索关键词**：白事铺、夜账、停灵室、无主尸骨、看守人、白纸灯。
- **复盘问题**：哪些夜账是人为伪造，哪些无法解释？义庄旧案和死亡预告如何关联？
