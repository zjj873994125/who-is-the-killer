# 夜走水路

## 故事正文

柳河镇靠水吃饭，也怕水。老人说，夜里不能坐最后一班渡船，因为最后一班不是给活人摆渡的。

失踪的是镇中老师周闻。他晚上十点从学校离开，监控拍到他走向渡口。渡口登记本上没有他的名字，但售票机吐出一张船票：柳河北岸至南岸，时间二十三点四十七分，票号 0000。

船夫说，最后一班九点半就停了。可周闻妻子收到一条语音，里面是水声和船桨声。周闻声音很低：“船上还有人，他们都不说话。”

第二天清晨，渡口台阶上有一串湿脚印，从河里走上来，又停在售票机前。脚印旁放着周闻的教案，纸张没有湿，像被人特意护着。教案夹着一张旧照片：年轻女子穿红嫁衣，站在渡船上，身后是柳河老桥。

调查员查到，三十年前柳河老桥塌过一次。塌桥那晚，一艘送亲船被卷入洪水，新娘和陪嫁亲属共七人失踪。镇志写成自然灾害，村里却一直传，说那船不该夜里过河。

周闻最近在带学生做地方史调查。他访问老人，查到塌桥并非天灾。老桥施工时偷工减料，镇干部收了钱。送亲船出事后，施工方和镇里把责任推给洪水，还把遇难者家属赶出镇子。

周闻的教案里写着一个名字：林玉娘。她就是当年的新娘。照片背后有一句话：夜走水路，莫问船钱。

售票机被拆检后，发现它没有联网，却在失踪当晚打印过票。票号 0000 是测试票，只能由管理员钥匙触发。钥匙在现任渡口管理员手里，而管理员是当年镇干部的儿子。

管理员承认，他发现周闻在查旧案，想吓唬他，伪造船票和湿脚印。但他坚称没有见过周闻。当晚他只把旧渡船解开缆绳，让它漂到河中央，没想到船回来时，船舱里多了一件湿嫁衣。

警方沿河搜寻，在旧桥墩下发现周闻。他还活着，被困在废弃桥洞里。周闻说，他确实上了一条船。船上坐着七个人，都穿着旧衣服，脸被水泡得发白。船到河中央时，有个新娘问他：“桥是谁修的？”

周闻醒来后就在桥洞里，身边放着一叠旧账本。账本记录了三十年前老桥工程回扣。显然有人把证据藏在桥洞多年，也有人借怪谈把周闻引过去。

管理员父亲留下遗书，承认老桥事故真相。他死前把账本藏在桥洞，却不敢公开。管理员知道账本存在，却不知道位置。他伪造怪谈，反而把周闻送到了真相旁边。

旧案重启后，柳河渡口停运整修。镇里在老桥遗址立了碑。碑上刻着七名遇难者的名字，包括林玉娘。

立碑那晚，售票机又打印出一张票。票号不是 0000，而是 0007。起点柳河北岸，终点空白。

票背面湿漉漉的，像刚从河水里捞出来。上面有一行娟秀的字：已到岸。

## 游戏化要点

- **核心怪谈**：夜里最后一班渡船会载走活人。
- **真相方向**：塌桥旧案、工程腐败、伪造怪谈和河边证据相互嵌套。
- **关键场景**：柳河渡口、售票机、旧桥墩、废弃桥洞、老渡船。
- **关键线索**：0000 船票、湿脚印、旧嫁衣、地方史教案、工程账本、0007 票。
- **检索关键词**：夜走水路、柳河老桥、林玉娘、0000、湿脚印、工程账本。
- **复盘问题**：周闻为什么会拿到不存在的船票？伪造怪谈的人和旧案真相是什么关系？
