# 纸新娘

## 故事正文

南桥镇有条老规矩，纸扎铺夜里不送货。白天送纸人，晚上送活人。这句话本是吓孩子的，直到镇东张家门口出现一具纸新娘。

纸新娘穿红嫁衣，盖红盖头，手腕上系着铜钱串。张家老太太开门时吓得跌坐在地，因为那嫁衣的样式，和二十年前失踪的女儿张小满出嫁那天一模一样。

张小满没有真的出嫁。二十年前，她和邻村青年私奔未遂，半路失踪。家里对外说她跟人跑了，族谱上把她名字划掉。可纸新娘送来的第二天，张家井里浮起一只红绣鞋，鞋底绣着“小满”两个字。

镇上开始传，说张小满回来成亲了。纸扎铺老板否认送过货，但账本上确实多了一笔订单：纸新娘一具，红衣全套，送张家老宅。下单人没有姓名，只写“回门”。

调查员来到张家。老宅门槛下压着一张黄纸，纸上写了四个生辰八字，其中一个属于张小满，另三个已经死亡。张家人说，这是二十年前请阴阳先生做的“断亲纸”，用来断掉张小满的姻缘和家门关系。

纸新娘不是第一具。调查发现，最近一个月，南桥镇有三户人家收到过纸扎物：一户收到纸婴，一户收到纸轿，一户收到纸马。三户都和当年张小满失踪有关。纸婴送到接生婆家，纸轿送到媒婆家，纸马送到村口车夫家。

接生婆已经中风，听见张小满的名字就哭。她女儿交出一封旧信，信里写张小满当年怀孕，被家里逼着嫁给一个死去的富户少爷，办阴婚冲喜。张小满不肯，夜里逃跑，被张家人和媒婆追到河边。

车夫的儿子说，父亲临死前常喊“轿子里没人，河里有人”。当年所谓私奔，是张家编出的说法。张小满没有离镇，她被塞进喜轿送去冥婚，途中挣扎逃出，掉进河里。家里怕丑事传开，把她失踪说成私奔。

纸扎铺账本的墨迹很新。老板承认有人半夜把钱和纸样塞进门缝，要求照旧式样制作。他没敢拒绝，因为纸样背面写着一行字：欠她一场婚，就还她一场婚。

调查员在张家祠堂找到关键证据。族谱被人重新翻开，张小满那一页的墨迹被水晕开，划掉的名字露了出来。页缝里夹着一枚铜钱，和纸新娘手腕上的铜钱一致。铜钱来自当年阴婚聘礼。

真相并非纸新娘杀人，而是张小满的弟弟张春生在复仇。他当年年幼，后来从老人闲话里拼出真相。母亲临死前交给他一只红绣鞋，他开始按当年参与者逐一送纸扎物，逼他们承认。

可张春生也说不清第一具纸新娘是谁放的。他只承认后三件纸扎物。第一具纸新娘的纸料很旧，不是现在的纸，红盖头里有水渍和河泥。纸新娘胸口贴着一张褪色婚书，上面新郎姓名空白，新娘写着张小满。

案子重启后，河道清淤，在旧桥墩下找到一具女性遗骸。遗骸手腕上缠着铜钱串，脚上少了一只红绣鞋。

张家为张小满重新立碑。下葬那天，纸新娘被烧给她。火苗烧到红盖头时，风忽然停了。灰里剩下一枚铜钱，没有烧黑，像刚从水里捞上来。

第二天，张家门口又出现一张小纸条：回门已毕。

## 游戏化要点

- **核心怪谈**：纸新娘送上门，牵出二十年前被掩盖的冥婚失踪案。
- **真相方向**：民俗恐怖外壳下是宗族遮丑、阴婚和亲属复仇。
- **关键场景**：张家老宅、纸扎铺、祠堂、旧桥、河道、接生婆家。
- **关键线索**：纸新娘、红绣鞋、断亲纸、纸扎订单、旧信、铜钱聘礼、空白婚书。
- **检索关键词**：纸新娘、张小满、回门、断亲纸、铜钱串、红绣鞋。
- **复盘问题**：张小满为什么被族谱除名？第一具纸新娘和后续纸扎物有什么区别？
