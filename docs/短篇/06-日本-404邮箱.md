# 404 邮箱

## 故事正文

青森私立大学旧论坛已经停用八年，服务器却还挂在信息中心角落。学生偶尔进去翻旧帖，把它当校园鬼故事素材。传说只要给 404@aomori-u.jp 发邮件，就会收到一封未来的退学通知。

大二学生藤原最先验证。他给 404 邮箱发了一句“你是谁”，三分钟后收到回复。邮件标题是“退学处理预通知”，正文写着：藤原悠介，因长期缺勤和精神异常，拟于六月十二日办理退学。落款日期是两周后。

同学以为他伪造截图。直到藤原真的开始缺课。他说自己每天醒来都会发现邮箱里多一封未读邮件，内容是他第二天要做的事，语气像教务处通知，又像某人在替他写人生。第七封邮件里写：请于凌晨三点前往旧图书馆四楼。

藤原去了，没再回来。

学校封锁消息，声称他休学回家。调查员进入旧论坛后台，发现 404 邮箱并不存在于当前邮件系统，却存在于八年前迁移前的备份账号里。账号备注是“异常退学学生归档”。

八年前，旧图书馆四楼发生过火灾，一名叫三浦玲奈的学生死亡。她生前被指控盗取试卷，学校发出退学通知。后来火灾调查草草结束，试卷泄露案也没人再提。

藤原失踪前曾在旧论坛发帖，标题是“八年前退学通知是不是假的”。他上传了一张旧邮件截图，里面显示三浦玲奈的退学通知发送时间，是她死亡后的第二天。

调查员调取旧图书馆门禁。藤原失踪当晚，门禁没有他的记录，但四楼电子阅览室有一台电脑在凌晨三点零四分登录了旧论坛。登录账号是三浦玲奈。

邮件服务器备份里保留了 404 邮箱的全部往来。调查员发现，404 邮箱并非灵异账号，而是当年教务处用来批量处理“问题学生”的隐藏邮箱。所有通过它发送的退学通知，都不会进入正式流程。三浦玲奈正是被这个邮箱提前“处理”的学生。

更深的线索在试卷泄露案里。真正盗取试卷的是一名教授的亲属，三浦玲奈发现后准备举报。学校为了压下丑闻，用 404 邮箱伪造退学流程，制造她精神不稳定的记录。火灾当晚，她被叫到旧图书馆四楼“谈话”。

藤原查到了这件事。他不是被鬼带走，而是被人约去旧图书馆。可监控只拍到他进入校园，没有拍到进入图书馆。四楼电脑旁留下他的学生证，桌面打开着一封新邮件。

邮件正文写：你找到我了，现在轮到你被删除。

警方在旧图书馆天花板夹层找到藤原。他还活着，脱水昏迷。旁边有一部旧笔记本电脑，自动运行脚本，定时从 404 邮箱发送恐吓邮件。操作者是当年信息中心管理员。他保留旧服务器多年，想用怪谈逼学校重查三浦案，却没想到藤原独自前来，被真正涉案的人困在夹层。

案子公开后，学校关闭旧论坛，三浦玲娜的处分被撤销。404 邮箱被删除，服务器硬盘被封存。

毕业季前，信息中心收到一封系统退信。发件人是 404@aomori-u.jp，收件人不存在。退信附件里是一张旧图书馆四楼照片，窗边站着一个女生，背对镜头，白衬衫被烟熏黑。

照片文件名是：未完成的退学通知。

## 游戏化要点

- **核心怪谈**：给 404 邮箱发信，会收到未来退学通知。
- **真相方向**：隐藏邮箱、伪造处分、旧火灾案和校园权力遮掩。
- **关键场景**：旧论坛、信息中心、旧图书馆四楼、电子阅览室、邮件服务器备份。
- **关键线索**：404 邮箱、未来退学通知、死亡后发送时间、隐藏账号备注、旧邮件往来、夹层电脑。
- **检索关键词**：404邮箱、退学通知、三浦玲奈、旧论坛、凌晨三点、异常归档。
- **复盘问题**：藤原为什么会相信邮件？404 邮箱如何成为旧案的遮掩工具？
