# 雨夜红伞

## 故事正文

风见坂十字路口没有遮雨棚，却总有人说那里有一把红伞。它只在深夜监控里出现，立在斑马线中央，伞面很低，看不见撑伞人的脸。

第一个报案的是出租车司机。他凌晨一点经过路口，看见红灯下站着一个撑红伞的女人。车灯扫过去时，女人没有影子。司机急刹车，后车追尾。交警调监控，却发现那晚气象记录显示无雨。

三天后，大学生岸田在同一路口失踪。他最后给朋友发了一张照片，照片里红伞挡住半个画面，伞沿下露出一截白色制服袖口。朋友以为是恶作剧，直到警方在路口绿化带里找到岸田的背包。

监控显示，岸田站在斑马线边看手机。凌晨一点十三分，画面左侧出现红伞。红伞靠近他，他抬头，然后跟着伞走进监控死角。十五秒后，红伞单独回来，停在斑马线中央。

调查员注意到一个细节：红伞每次出现的位置，都在斑马线第四条白线附近。十年前，这里发生过一起校车事故。一名穿白色制服的高中女生被卷入车底，雨伞被压在斑马线上。事故后，路口重铺，白线位置略有偏移，原来的事故点被新的标线盖住。

岸田的父亲正是当年校车司机。事故档案写明司机无责，因为女生闯红灯。但旧新闻照片里，信号灯显示的是绿灯。后来照片从网站删除，保留下来的只有低清截图。

警方查到，当年路口信号灯维护由一家外包公司负责。事故当天，信号灯曾短暂故障，南北向和东西向同时出现绿灯。外包公司为了免赔，修改了维护日志。岸田父亲也签下补充陈述，说自己看见女生闯灯。

红伞像是在挑选与旧案有关的人。出租车司机当年是追尾车辆驾驶员，替校车司机作证说“雨太大看不清”。岸田作为司机儿子并不知情，却在父亲旧物中找到一张雨伞照片。他把照片发到网上，留言说“都市传说不过是事故家属敲诈”。

失踪第七天，岸田的手机在路口雨水井里被找到。手机里有一段视频。画面摇晃，像是有人被牵着走。红伞在前方，伞下没有脚。背景能听见少女声音：“那天是绿灯，对吧？”

警方重新检测路口地下排水井，发现井壁有一处被封死的检修口，里面放着一束干枯红伞和多份复印材料。材料来自事故家属多年申诉，包括被篡改的信号维护记录。

岸田没有死。他被发现时昏迷在旧信号控制箱旁，手里攥着父亲当年的补充陈述。醒来后，他只说一句话：红伞不是来找我，是让我把这个交出去。

旧案重启。外包公司负责人被调查，岸田父亲承认作伪证。风见坂十字路口重新立了事故纪念牌。

纪念牌落成那天，下了一场小雨。有人在牌前放了一把红伞。雨停后，伞不见了。

监控最后一次拍到它，是凌晨一点十三分。红伞停在斑马线中央，伞面慢慢抬起，像有人终于看清了信号灯。

## 游戏化要点

- **核心怪谈**：无雨深夜出现的红伞会带走路口行人。
- **真相方向**：旧交通事故、信号灯故障、伪证和家属申诉构成真相链。
- **关键场景**：风见坂路口、监控死角、雨水井、旧信号控制箱、事故纪念牌。
- **关键线索**：无雨记录、第四条白线、旧新闻截图、维护日志、手机视频、补充陈述。
- **检索关键词**：红伞、风见坂、第四条白线、01:13、信号维护、补充陈述。
- **复盘问题**：红伞为什么只在特定位置出现？岸田为什么会被选中？
