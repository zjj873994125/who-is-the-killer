# 回门饭

## 故事正文

青瓦巷办喜事讲究回门饭。新娘第三天带新郎回娘家，娘家摆席，桌上几副碗筷就代表几口人。沈家那天摆了十三副碗筷，可进门的明明只有十二个人。

多出来的那副碗筷摆在主桌西侧，碗里盛着白饭，饭上插着一双筷子。新娘沈月看见后脸色发白，说那是给死人吃的。母亲却说是手滑摆错，转身就把碗撤了。

饭吃到一半，席上所有手机同时收到一张照片。照片拍的是沈家厨房，灶台前站着一个穿旧式蓝布衣的女人，背对镜头，正往锅里放盐。照片时间是十分钟前，可厨房里那时只有沈母一人。

当晚，新郎腹痛送医，查出轻微中毒。其他人无事。警方检查剩菜，发现只有新郎碗里的莲藕汤含有农药残留。沈家人一口咬定是意外，直到调查员在垃圾桶里找到第十三副碗筷，筷尖沾着同样的农药。

沈家老宅里有个忌讳：吃饭不能坐西侧。二十年前，沈月的姐姐沈秋在回门当天死在那个位置。档案写她与丈夫争吵后服毒自杀。沈家从此不提她，家里旧相册也被剪掉她的照片。

调查员发现这次回门饭的菜单，和二十年前完全一样：莲藕汤、红烧鱼、扣肉、炒青菜、甜酒圆子。每道菜都对应当年席上的一个人。二十年前席上共有十三人，除了沈秋，其他十二人今天都在。

沈月说，她结婚前一晚梦见姐姐。姐姐坐在西侧位置，问她：“你知道我那碗汤是谁盛的吗？”醒来后，她在嫁妆箱里找到一张旧菜单，背面写着十二个名字。

调查员顺着名单查问。二十年前，沈秋并非自杀。她发现丈夫和沈家堂妹私通，准备离婚。沈家怕丢脸，劝她忍下。回门饭那天，堂妹在莲藕汤里下药，想让她“病一场”，没想到剂量过重。沈家人为了保住婚事和脸面，把死亡说成自杀。

这次有人复制当年的菜和座位，把毒下到新郎碗里。嫌疑最大的是沈月，因为新郎最近也被发现出轨。可沈月没有下毒。监控显示，下毒的是沈母。她害怕沈月重走姐姐老路，想用轻微中毒吓退新郎，也逼沈家人想起当年的罪。

那么第十三副碗筷是谁摆的？沈母否认。厨房照片也无法解释。照片不是手机拍摄，而是来自沈家十年前拆掉的老式监控，文件格式早已停用。技术员恢复硬盘，发现二十年前回门饭当天确实有监控。画面里，沈秋喝汤前，曾把自己的碗推给堂妹，堂妹又偷偷换了回来。

真正导致沈秋死亡的不是一人，而是整桌人的沉默。

案子公开后，沈家重新为沈秋立照。沈月取消婚礼后续仪式。她把第十三副碗筷收进姐姐牌位前，不再插筷，只盛了一碗热饭。

那晚，沈月收到最后一张照片。照片里主桌西侧空着，碗筷摆得端正。背后墙上的旧相框中，沈秋第一次露出完整的脸。

## 游戏化要点

- **核心怪谈**：回门饭多出第十三副碗筷，引出二十年前同席死亡案。
- **真相方向**：婚姻家丑、集体沉默、复刻菜单和轻度投毒形成推理链。
- **关键场景**：沈家老宅、回门饭主桌、厨房、嫁妆箱、旧监控硬盘。
- **关键线索**：第十三副碗筷、插筷白饭、旧菜单、十二个名字、农药残留、厨房照片。
- **检索关键词**：回门饭、沈秋、西侧座位、莲藕汤、旧菜单、十三副碗筷。
- **复盘问题**：这次中毒和二十年前死亡有什么对应关系？第十三副碗筷为什么必须出现？
