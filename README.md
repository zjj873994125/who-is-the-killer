# 有诡电梯 Demo

《有诡电梯》是一个本地网页推理小游戏 demo。首版实现了序章和第一章闭环：玩家进入刑侦档案系统，接入特案组通讯，进入电梯现场勘查，通过关键词检索归档线索，并在复盘区验证电梯运行矛盾。

## 技术栈

- Vue 3 + TypeScript
- Vite
- Vue Router
- Element Plus
- Less
- GSAP
- Vitest

## 本地运行

```bash
npm install
npm run dev -- --host 127.0.0.1
```

默认访问：

```text
http://127.0.0.1:5173/
```

## 验证命令

```bash
npm test
npm run build
```

## 当前功能

- 内容警告与档案系统入口
- 路由式工作台
- 案件区：序章、立案材料、进入勘查区
- 勘查区：电梯现场、明确点位、观察记录
- 线索区：关键词检索、已归档证据、线索详情
- 复盘区：可提前进入查看规则；归档全部 15 条线索后开放证据组合选择，验证电梯运行矛盾
- 故事区：复盘成功后解锁，展示第一章完整结案故事和生成图片卷宗
- 后续档案：第二章《见鬼十法》占位
- localStorage 本地存档
